Primitives, Fonctions Logarithmiques et Exponentielles

Comprendre et manipuler les fonctions fondamentales de l'analyse
Point essentiel Niveau Terminale S

📌 Introduction

Les primitives, les fonctions logarithmiques et exponentielles sont des notions fondamentales en analyse mathématique. Les primitives constituent la base du calcul intégral, tandis que les fonctions logarithmiques et exponentielles apparaissent dans de nombreux phénomènes naturels comme la croissance, la décroissance et les évolutions proportionnelles. Ces concepts sont essentiels pour modéliser des situations réelles en physique, biologie, économie et bien d'autres domaines.

1. Primitives d'une Fonction

Définition et méthodes de calcul

Définition d'une primitive

Une fonction F est une primitive de la fonction f sur un intervalle I si, pour tout x de I, F'(x) = f(x).

Si F est une primitive de f, alors toute fonction de la forme F + C, où C est une constante réelle, est aussi une primitive de f.

On note l'ensemble des primitives de f par :

∫ f(x) dx = F(x) + C, où C est une constante réelle

Primitives usuelles

Fonction f(x)
Primitive F(x)

Linéarité des primitives

Les primitives respectent les propriétés de linéarité :

∫ [f(x) + g(x)] dx = ∫ f(x) dx + ∫ g(x) dx

∫ k·f(x) dx = k·∫ f(x) dx (où k est une constante)

Exemple : Calcul de primitives

Calculons une primitive de f(x) = 3x² + 2x - 5

∫ (3x² + 2x - 5) dx = ∫ 3x² dx + ∫ 2x dx - ∫ 5 dx

= 3·∫ x² dx + 2·∫ x dx - 5·∫ dx

= 3·(x³/3) + 2·(x²/2) - 5x + C

= x³ + x² - 5x + C

Primitive d'une fonction composée

Pour certaines fonctions composées, on peut utiliser la formule suivante :

∫ u'(x)·v(u(x)) dx = ∫ v(u)·du

Cette méthode est particulièrement utile pour les fonctions de la forme u'(x)·f(u(x)).

Exemple : Primitive d'une fonction composée

Calculons une primitive de f(x) = (2x+1)·ex²+x

Posons u(x) = x² + x, alors u'(x) = 2x + 1

f(x) = u'(x)·eu(x)

∫ f(x) dx = ∫ u'(x)·eu(x) dx = ∫ eu du = eu + C

= ex²+x + C

Exercices d'application
1. Calculer une primitive de f(x) = 2x³ - 3x + 4

∫ (2x³ - 3x + 4) dx = 2·∫ x³ dx - 3·∫ x dx + 4·∫ dx

= 2·(x⁴/4) - 3·(x²/2) + 4x + C

= (x⁴/2) - (3x²/2) + 4x + C

2. Déterminer la primitive F de f(x) = cos(2x) telle que F(0) = 3

∫ cos(2x) dx = (1/2)·∫ cos(2x)·2 dx = (1/2)·sin(2x) + C

F(0) = (1/2)·sin(0) + C = 0 + C = 3, donc C = 3

F(x) = (1/2)·sin(2x) + 3

2. Fonction Logarithme Népérien

Définition, propriétés et représentation graphique

Définition et domaine de définition

La fonction logarithme népérien, notée ln, est définie comme la primitive de la fonction 1/x qui s'annule en 1 :

ln(x) = ∫₁x (1/t) dt

Domaine de définition : ]0, +∞[

La fonction ln est l'inverse de la fonction exponentielle : si y = ln(x), alors ey = x

Propriétés fondamentales

Pour tous réels strictement positifs a et b :

ln(a·b) = ln(a) + ln(b)

ln(a/b) = ln(a) - ln(b)

ln(an) = n·ln(a) (n réel)

ln(1) = 0

ln(e) = 1

Dérivée et primitive

Dérivée : Pour tout x > 0, (ln(x))' = 1/x

Primitive : ∫ ln(x) dx = x·ln(x) - x + C

Étude de la fonction ln

Variations : La fonction ln est strictement croissante sur ]0, +∞[

Limites :

limx→0⁺ ln(x) = -∞

limx→+∞ ln(x) = +∞

Convexité : La fonction ln est concave sur ]0, +∞[ (sa dérivée seconde est négative)

Représentation graphique

[Courbe représentative de la fonction ln]

La courbe passe par le point (1,0), admet l'axe des ordonnées comme asymptote verticale et croît très lentement vers +∞

Équations et inéquations avec ln

Pour résoudre des équations ou inéquations avec ln, on utilise souvent les propriétés suivantes :

  • ln(a) = ln(b) ⟺ a = b (car ln est injective)
  • ln(a) > ln(b) ⟺ a > b (car ln est strictement croissante)
Exemple : Résolution d'équation

Résolvons l'équation ln(x+1) - ln(x) = ln(2)

ln((x+1)/x) = ln(2)

(x+1)/x = 2

x+1 = 2x

1 = x

La solution est x = 1

Exemple : Résolution d'inéquation

Résolvons l'inéquation ln(x) > 2

Comme ln est strictement croissante : ln(x) > 2 ⟺ x > e²

L'ensemble des solutions est ]e², +∞[

Exercices d'application
1. Résoudre l'équation ln(x²) = 2ln(x)

ln(x²) = 2ln(x)

2ln(x) = 2ln(x)

Cette équation est vérifiée pour tout x > 0

L'ensemble des solutions est ]0, +∞[

2. Résoudre l'inéquation ln(x+1) < ln(3x)

ln(x+1) < ln(3x)

Comme ln est strictement croissante : x+1 < 3x

1 < 2x

1/2 < x

L'ensemble des solutions est ]1/2, +∞[ ∩ ]0, +∞[ = ]1/2, +∞[

3. Fonction Exponentielle

Définition, propriétés et représentation graphique

Définition et domaine de définition

La fonction exponentielle, notée exp ou ex, est l'unique fonction f telle que :

  • f'(x) = f(x) pour tout réel x
  • f(0) = 1

Domaine de définition : ℝ (ensemble des réels)

La fonction exponentielle est l'inverse de la fonction logarithme népérien : si y = ex, alors ln(y) = x

Propriétés fondamentales

Pour tous réels a et b :

ea+b = ea·eb

ea-b = ea/eb

(ea)b = ea·b

e0 = 1

e1 = e ≈ 2,718

Dérivée et primitive

Dérivée : Pour tout x réel, (ex)' = ex

Primitive : ∫ ex dx = ex + C

Étude de la fonction exponentielle

Variations : La fonction exponentielle est strictement croissante sur ℝ

Limites :

limx→-∞ ex = 0

limx→+∞ ex = +∞

Convexité : La fonction exponentielle est convexe sur ℝ (sa dérivée seconde est positive)

Représentation graphique

[Courbe représentative de la fonction exponentielle]

La courbe passe par le point (0,1), admet l'axe des abscisses comme asymptote horizontale quand x tend vers -∞ et croît très rapidement vers +∞

Équations et inéquations avec l'exponentielle

Pour résoudre des équations ou inéquations avec l'exponentielle, on utilise souvent les propriétés suivantes :

  • ea = eb ⟺ a = b (car exp est injective)
  • ea > eb ⟺ a > b (car exp est strictement croissante)
Exemple : Résolution d'équation

Résolvons l'équation e2x = ex+1

Comme la fonction exponentielle est injective :

2x = x+1

x = 1

La solution est x = 1

Exemple : Résolution d'inéquation

Résolvons l'inéquation ex < 3

Comme la fonction exponentielle est strictement croissante :

ex < 3 ⟺ x < ln(3)

L'ensemble des solutions est ]-∞, ln(3)[

Exercices d'application
1. Résoudre l'équation ex²-4 = 1

ex²-4 = 1

ex²-4 = e0

x²-4 = 0

x² = 4

x = ±2

Les solutions sont x = -2 et x = 2

2. Résoudre l'inéquation ex + e-x ≥ 2

ex + e-x ≥ 2

Posons f(x) = ex + e-x

f'(x) = ex - e-x

f'(x) = 0 ⟺ ex = e-x ⟺ e2x = 1 ⟺ 2x = 0 ⟺ x = 0

f'(x) > 0 pour x > 0 et f'(x) < 0 pour x < 0

Donc f admet un minimum en x = 0 et f(0) = e0 + e-0 = 1 + 1 = 2

Ainsi, f(x) ≥ 2 pour tout x réel

L'ensemble des solutions est ℝ

4. Applications et Modélisation

Utilisation des fonctions logarithmiques et exponentielles

Croissance exponentielle

Un phénomène suit une croissance exponentielle lorsque son taux de variation est proportionnel à sa valeur actuelle.

Mathématiquement, si y(t) représente la quantité au temps t, alors :

y'(t) = k·y(t) (où k > 0 est une constante)

La solution de cette équation différentielle est : y(t) = y₀·ekt (où y₀ est la valeur initiale)

Exemple : Croissance d'une population

Une population de bactéries double toutes les 3 heures. Si la population initiale est de 1000 bactéries, combien y aura-t-il de bactéries après 9 heures ?

La population suit une loi exponentielle : N(t) = N₀·ekt

Après 3 heures, N(3) = 2·N₀, donc N₀·e3k = 2·N₀

e3k = 2, donc 3k = ln(2), et k = ln(2)/3

Après 9 heures : N(9) = N₀·e9k = N₀·e9·ln(2)/3 = N₀·e3·ln(2) = N₀·(eln(2))3 = N₀·2³ = 1000·8 = 8000

Il y aura 8000 bactéries après 9 heures.

Décroissance exponentielle

Un phénomène suit une décroissance exponentielle lorsque son taux de variation est proportionnel à sa valeur actuelle, mais dans le sens opposé.

y'(t) = -k·y(t) (où k > 0 est une constante)

La solution de cette équation différentielle est : y(t) = y₀·e-kt (où y₀ est la valeur initiale)

Exemple : Désintégration radioactive

La demi-vie d'un élément radioactif est de 5730 ans (temps nécessaire pour que la moitié des atomes se désintègrent). Quelle proportion de l'élément restera après 11460 ans ?

La quantité d'élément suit une loi de décroissance exponentielle : N(t) = N₀·e-kt

Après 5730 ans, N(5730) = N₀/2, donc N₀·e-5730k = N₀/2

e-5730k = 1/2, donc -5730k = ln(1/2) = -ln(2), et k = ln(2)/5730

Après 11460 ans : N(11460) = N₀·e-11460k = N₀·e-11460·ln(2)/5730 = N₀·e-2·ln(2) = N₀·(e-ln(2))2 = N₀·(1/2)² = N₀/4

Il restera 1/4 (soit 25%) de l'élément après 11460 ans.

Intérêts composés

Lorsqu'un capital est placé à intérêts composés, sa valeur suit une croissance exponentielle.

Si un capital C₀ est placé à un taux d'intérêt annuel r (exprimé en décimal), sa valeur après t années est :

C(t) = C₀·(1+r)t

Si les intérêts sont composés continuellement, la formule devient :

C(t) = C₀·ert

Exemple : Calcul d'intérêts composés

Un capital de 1000€ est placé à un taux d'intérêt annuel de 5%. Quelle sera sa valeur après 10 ans ?

C(10) = 1000·(1+0,05)10 = 1000·1,0510 ≈ 1000·1,629 = 1629€

Le capital vaudra environ 1629€ après 10 ans.

Échelle logarithmique

L'échelle logarithmique est utilisée pour représenter des données qui couvrent plusieurs ordres de grandeur.

Applications courantes :

  • Échelle de Richter pour les séismes (logarithme de l'amplitude)
  • Décibels pour l'intensité sonore (logarithme de la puissance)
  • pH pour l'acidité (logarithme négatif de la concentration en ions H⁺)
Exemple : Échelle de pH

Le pH est défini par : pH = -log₁₀[H⁺], où [H⁺] est la concentration en ions hydrogène en mol/L.

Si une solution a une concentration [H⁺] = 10⁻³ mol/L, son pH est :

pH = -log₁₀(10⁻³) = -(-3) = 3

Une solution de pH = 3 est acide (pH < 7).

Exercices d'application
1. Un capital de 2000€ est placé à un taux d'intérêt annuel de 3%. Au bout de combien d'années ce capital aura-t-il doublé ?

On cherche t tel que C(t) = 2·C₀

2000·(1+0,03)t = 2·2000

(1,03)t = 2

t·ln(1,03) = ln(2)

t = ln(2)/ln(1,03) ≈ 0,693/0,0296 ≈ 23,4

Le capital aura doublé au bout d'environ 23,4 ans.

2. La population d'une ville augmente de 2% par an. En combien d'années la population aura-t-elle augmenté de 50% ?

On cherche t tel que P(t) = 1,5·P₀

P₀·(1+0,02)t = 1,5·P₀

(1,02)t = 1,5

t·ln(1,02) = ln(1,5)

t = ln(1,5)/ln(1,02) ≈ 0,405/0,0198 ≈ 20,5

La population aura augmenté de 50% au bout d'environ 20,5 ans.

Conclusion : Primitives, Fonctions Logarithmiques et Exponentielles

Synthèse et applications

Points clés à retenir

📌
Primitives

Les primitives sont essentielles pour le calcul intégral et permettent de déterminer l'aire sous une courbe.

📌
Fonction logarithme

La fonction ln est strictement croissante sur ]0, +∞[, concave, et vérifie ln(a·b) = ln(a) + ln(b).

📌
Fonction exponentielle

La fonction exponentielle est strictement croissante sur ℝ, convexe, et vérifie ea+b = ea·eb.

📌
Applications

Ces fonctions modélisent de nombreux phénomènes naturels et économiques comme la croissance, la décroissance et les intérêts composés.

Relations fondamentales

Les fonctions logarithmiques et exponentielles sont intimement liées :

ln(ex) = x pour tout x réel

eln(x) = x pour tout x > 0

Ces relations font de ces fonctions des outils puissants pour résoudre des équations et modéliser des phénomènes complexes.
